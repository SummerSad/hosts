=======================
Hosts file to block spy
=======================

How to use
==========

Simply copy data from `spy
<https://raw.githubusercontent.com/SummerSad/hosts/master/spy>`_
to host file

Flush dns ::

        ipconfig /flushdns

Sources of hosts
================
